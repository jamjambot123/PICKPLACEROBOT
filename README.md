# 반도체 픽앤플레이스 갠트리 진동 제어 시스템 (Digital Twin Simulator)

## 📌 프로젝트 개요
반도체 패키지 픽앤플레이스 공정에서 갠트리의 고속 이송 시 발생하는 **잔류 진동(Residual Vibration)**을 억제하여, 정착 시간(Settling Time)을 단축하고 최종적으로 **생산성(UPH, Units Per Hour)을 극대화**하기 위한 디지털 트윈 시뮬레이터입니다.

기존 실물 하드웨어(STM32, NEMA 스텝모터, MPU6050 IMU)를 통해 검증하려던 계획을 **물리 엔진 기반의 웹 시뮬레이션**으로 전환하여, 다양한 모션 프로파일의 성능을 브라우저 상에서 즉각적이고 정량적으로 비교할 수 있도록 구현했습니다.

## 🎯 주요 목표 (User's Goal)
1. **기술 검증**: 사다리꼴(Trapezoidal), 비대칭 S-curve, 그리고 **인풋쉐이핑(Input Shaping, ZVD)**을 적용했을 때의 잔류 진동 및 정착 시간 차이를 직관적이고 정량적으로 증명.
2. **최적화 도구**: AI(입자 군집 최적화, PSO)를 활용하여 각 프로파일의 한계 성능(최대 UPH)을 자동으로 탐색.
3. **가시성**: 3D 환경에서 제어 적용(Shaped) 로봇과 제어 미적용(Unshaped/Ghost) 로봇을 동시에 병치하여 직관적인 성능 차이 시연.

## 🚀 주요 기능 (Features)

### 1. 물리 엔진 (2-Mass Model)
- **Perfect Servo + Flexible Load**: 모터(Carriage)는 지령 위치를 완벽히 추종하며, 끝단의 로봇 팔(Load)은 질량-스프링-댐퍼 시스템으로 모델링되어 기계적 진동을 현실적으로 모사.
- **Payload 스위칭**: 화물 적재 여부에 따라 시스템 질량이 변동되며 고유진동수(fn)가 실시간으로 달라지는 현상(Robustness 테스트) 구현.

### 2. 모션 프로파일 생성기
- **Trapezoidal (사다리꼴)**: 기준 프로파일 (저크 무제한, 최대 진동 유발)
- **AS-Curve (비대칭 S-curve)**: 가속/감속 구간의 저크를 독립적으로 제어하여 기구부 충격 완화
- **Input Shaping (ZV, ZVD, EI)**: 시스템의 고유진동수와 감쇠비를 기반으로 입력 신호를 성형하여 잔류 진동을 근본적으로 상쇄

### 3. AI 자동 튜닝 (PSO Optimizer)
- **목적 함수**: 잔류 진동이 허용 오차(±5μm) 이내로 수렴하는 정착 시간을 최소화하여 UPH 극대화.
- 사용자가 수동으로 파라미터를 찾을 필요 없이, AI가 Vmax, Amax, Beta, Gamma 값을 수백 번 시뮬레이션하여 최적값을 도출.

### 4. 3-프로파일 정량 비교 (Comparison Modal)
- **동일 조건 비교**: 통제된 변인(동일 Vmax, Amax) 하에서 기술별 효과 증명.
- **AI 최적화 비교**: AI가 각 프로파일의 한계 성능까지 튜닝한 결과를 비교하여, Input Shaping 도입 시의 최종적인 산업적 기대 효과(UPH 증가율 등)를 테이블과 CSV로 출력.

### 5. 전문적 시각화 (MATLAB / Simulink 스타일 UI)
- Three.js 기반 3D 디지털 트윈 환경.
- 실시간 위치, 속도, 가속도/저크, 잔류 진동 차트 제공 (Chart.js).
- 재생 속도 제어(0.1x ~ 3x)를 통해 찰나의 진동 현상을 슬로우 모션으로 관찰 가능.

## 📂 프로젝트 구조
- `index.html`: 메인 대시보드 UI (Config, 3D 뷰어, 차트 캔버스 등)
- `style.css`: 전문적인 엔지니어링 툴 느낌의 라이트/다크 테마 스타일링
- `js/motion-profile.js`: 사다리꼴 및 비대칭 S-curve 프로파일 수학적 궤적 생성기
- `js/input-shaping.js`: ZV/ZVD 쉐이퍼 연산 및 2-Mass 물리 엔진 시뮬레이션 연산 로직
- `js/ai-optimizer.js`: PSO 기반 파라미터 튜닝 최적화 알고리즘
- `js/scene.js`: Three.js를 이용한 갠트리 및 로봇팔 3D 시각화 (Main & Ghost 렌더링)
- `js/charts.js`: Chart.js 기반 실시간 데이터 시각화
- `js/app.js`: 전체 시스템 상태 관리, UI 이벤트 바인딩 및 시뮬레이션 루프 (컨트롤러 역할)

- 향후 추가 개발이나 수정을 진행할 경우, 가장 중요한 것은 **"학술적/산업적 검증 도구로서의 무결성"**입니다.
